# npm-badges
Display a npm badge as in example(png image)

## API

`npm-badge.php?name={npm-library-name}`
### Parameters
-   `name` **[string]** NPM library name

### Demo online

`http://www.vincenzodevivo.com/lab/npm-badges/npm-badge.php?name=super-mem`
